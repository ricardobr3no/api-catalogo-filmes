/filmes

Rota de filmes

/filmes get post

get

Retorna uma lista de filmes

post

Entre com as informaçoes para adicionar um novo filme

get /filmes

Retorna uma lista de filmes

  • Request
  • Response

Query Parameters

  • titulo: (string)

    titulo de algum filme

    Example:

    avatar
  • genero: (string)

    um genero do filme

    Example:

    horror
  • ano: (number)

    ano de lançamento do filme

    Example:

    1997

HTTP status code 200

Body

Media type: application/json

Type: any

Example:

[
  {
    "id": 1,
    "titulo": "avatar",
    "generos": ["acao", "aventura"],
    "diretor": "james cameron",
    "ano": 2009,
    "classificao": "14"
  }, 
  {
    "id": 2,
    "titulo": "velozes e furiosos 4",
    "generos": ["acao", "aventura"],
    "diretor": "james gunn",
    "ano": 2013,
    "classificao": "14"
  }

HTTP status code 404

Body

Media type: application/json

Type: any

Example:

[]

post /filmes

Entre com as informaçoes para adicionar um novo filme

  • Request
  • Response

Body

Media type: application/json

Type: object

Properties
  • titulo: required(string)
  • ano: required(int)
  • sinopse: (string)
  • classificao: required(int)
  • generoId: required(int)
  • diretorId: required(int)

Example:

{
  "titulo": "avatar",
  "generoId": [1, 2],
  "diretorId": 1,
  "ano": 2009,
  "classificao": "14"
  "sinopse": "sinopes do filme..."
}

HTTP status code 201

Body

Media type: application/json

Type: any

Example:

{"message": "filme adicionado com sucesso!" }
